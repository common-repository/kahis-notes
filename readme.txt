== Description ==

For me, it's place to leave notes about future articles, possible topics, ToDos connected with that particular WP site.

Nothing more to explain. [See the screenshot](http://kahi.cz/wordpress/wp-content/images/kahis-notes-plugin/notes-screen.png). Making notes inside your WordPress, as simple as possible.

For more information, support, giving feedback or anything - see the [official plugin's site](http://kahi.cz/wordpress/wp-notes-plugin/).

= Requirements =

**PHP 5** on your server is necessary to run this plugin.
